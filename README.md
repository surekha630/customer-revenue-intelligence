# Customer Revenue Intelligence Analysis

## Project Overview
This project analyzes transactional retail data to uncover the key drivers of revenue, customer purchasing behavior, and business performance.

The analysis transforms raw data into strategic insights that support executive decision-making.

---

## Business Problem
Retail organizations generate massive amounts of sales data but often lack clarity on:

- What drives revenue?
- Which customers matter most?
- How purchasing behavior impacts profitability
- Where growth opportunities exist
